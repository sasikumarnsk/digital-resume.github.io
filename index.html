WEB APPLICATION FOR STUDENT MONITORING SYSTEM (Spring Boot - Java)
This project has a web portal from where the faculties can enable the attendance in any of the two modes,
  1. Single attendance mode
  2. Continuous monitoring mode
Single attendance mode : This is similar to the standard roll call method , the student is checked only once.
Continuous attendance mode : In this mode the student will be checked in random intervals and report will be generated with the collective data.

Features :
  • Faculty can view attendance in realtime.
  • Faculty can generate report and it will be saved to database, the report can be downloaded as csv
  • Faculty can view analysis of the class
  • Students can view their realtime attendance percentage
  • NoticeBoard feature is available with priority modes.

AUTOMATED FINGERPRINT BASED ATTENDANCE SYSTEM VIA INTRANET (Django - Python)

The project is to reduce the time consumed while waiting in queue to authenticate and to reduce the initial cost of implementation as much as possible and to create a authenticating system which is easily scalable to any number of users without requiring more machines which promotes social distancing by not gathering around the same place and to create a robust authentication system with user friendliness. This project uses android-API, Django rest framework as backend and an android application, which is to be installed in user’s device.

After connecting to the authentication network, the android application can interact with authentication server and if the authentication slot is open, the user can authenticate him/her with the use of android biometric prompt and will be updated to the master database, the system can also generate automatic reports. And the system is designed to be flexible to many special occasions and any automated tasks and features can be easily implemented.
